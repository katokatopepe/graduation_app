## サービス概要
プロ野球観戦の結果を記録するサービス（仮称：Blog-Baseball Log）
　―サービス利用者が、プロ野球の公式戦を現地（野球場）で観戦した際の結果を記録する
　―自分が応援している"推しチーム"の対戦成績を記録するだけでなく、"推し選手"の成績等の記録を可能とする

## このサービスへの思い・作りたい理由
私は長年プロ野球のファンとして、毎年数多くの試合を観戦していますが、「あの試合はどのような内容だったか」「今年の戦績はどうだったか」と、過去に観戦した試合の結果を振り返りたい場面において、すぐに確認できるツールがなく、不便さを感じてきました。
また、近年のプロ野球における観客動員数は増加傾向にあり、従来とは異なり「推し活」の一環として球場を訪れるファンも増えています。
そうした中で、私と同様に、自身が過去に観戦した試合を振り返りたいと考えるファンは少なくないのではないかと感じ、本サービスの開発に着手することとしました。

## ユーザー層について
現地（野球場）で頻繁に観戦するプロ野球ファン
　―あくまで「現地観戦」の記録なので、世代は問わず頻繁に球場に訪れるファンを想定

## サービスの利用イメージ
① 観戦記録をサービス上に入力
② 推しチームのこれまでの対戦成績を表示（対戦相手・球場別等のカテゴリ別での表示を想定）
③ 過去訪問した試合を、自身が撮影した写真・動画や、動画サイト（YouTube等）のリンクより動画を見て、振り返りが可能
④ 観戦時の推し選手の戦績も入力・確認が可能

## ユーザーの獲得について
SNSでの拡散
　―プロ野球に関連するワードを＃にして、X（旧Twitter）やInstagramで発信


## サービスの差別化ポイント・推しポイント
① 自身が撮影した写真・動画や、動画サイトとの連動
現状リリースされている類似のサービスは対戦成績のみの記録が可能であるが、本サービスでは自身が撮影した写真・動画や、動画サイト（YouTube等）を試合記録に紐づけることで、戦績だけでなく様々な観点から思い出を記録することが可能となる
② 推し選手の記録
現状リリースされている類似のサービスでは推しチームの記録のみが可能であるが、本サービスでは事前に推し選手を登録し、推し選手の現地観戦時の結果を表示することを可能とする

## 機能候補
【MVPリリース】
・ユーザー登録（メールアドレス・ユーザ－名・パスワード・推しチーム）
・観戦成績記録（対戦相手・観戦球場・勝敗・スコア・先発投手・写真および動画・動画サイトURL等）および閲覧
・対戦結果表示（対戦相手別・観戦球場別・勝敗別等）

【本リリース（MVPリリースとの差異のみ記載）】
・ユーザー登録（メールアドレス・ユーザ－名・パスワード・推しチーム・推し選手）
・推し選手記録（試合結果・写真および動画・動画サイトURL等）および閲覧
・推し選手結果表示（対戦相手別・観戦球場別等）
